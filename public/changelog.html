バージョン管理

ver 6.0 (2025/08/13)

  • CI/CDパイプラインを構築し、GitHubへのプッシュをトリガーとしたテストとデプロイの自動化を実現。

ver 5.1 (2025/08/12)

  • バックエンドのmain.pyに残っていたバグ（feedparserライブラリの不正な引数）を修正し、ニュースフィードが安定して表示されるように最終修正。
  • 静的ツールマップと動的ニュースフィードの機能を1つのバックエンドに統合する最終コードを確定。

ver 5.0 (2025/08/12)

  • UI/UXを全面的に改善（レスポンシブデザイン対応）。
  • ヘッダーとCSSを共通ファイル化し、メンテナンス性を向上。

ver 4.0 (2025/08/12)

  • Cloud Runのコールドスタート対策として、最小インスタンスを1に設定。
  • Cloud SchedulerとCloud Runのタイムアウト時間を延長し、安定性を向上。

ver 3.0 (2025/08/11)

  • キャッシュ機能（Cloud Scheduler, Firestore）を導入し、表示の高速化とAPI利用の効率化を実現。
  • Googleトレンドの不安定さを解消するため、トレンド機能を「注目企業ランキング」に変更。

ver 2.0 (2025/08/11)

  • ニュースの自動カテゴリ分類機能を追加。
  • AIによる分析を、単なる要約から「PM視点での重要性」分析に変更。
  • UIをダッシュボード形式に刷新し、カテゴリフィルターを追加。

ver 1.0 (2025/08/10)

  • 主要なAIニュースサイトのRSSフィードから最新情報を収集する基本機能を実装。
  • バックエンドをCloud Run、フロントエンドをFirebase Hostingで構築。
